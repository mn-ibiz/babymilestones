# P1-E12-S04 — Sign-in / sign-up entry points

**Status:** Ready for development
**Phase:** P1 — Foundation + Wallet + Parent Account
**Epic:** P1-E12 — Marketing & Landing (public route group)
**Estimate:** 1–3 days

## Job To Be Done

As a visitor, I want a clear way to sign in or create an account from anywhere on the marketing site.

## Acceptance Criteria

- AC1: Header: "Sign in" + "Sign up" CTAs visible on all public pages.
- AC2: After auth, redirect honours intended destination (e.g., back to `/book/talent`).
- AC3: Auth UI uses the parent flow (phone + PIN).
