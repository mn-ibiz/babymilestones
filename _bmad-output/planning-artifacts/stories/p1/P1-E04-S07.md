# P1-E04-S07 — Bank transfer top-up (admin-confirmed)

**Status:** Ready for development
**Phase:** P1 — Foundation + Wallet + Parent Account
**Epic:** P1-E04 — Payments Adapter
**Estimate:** 1–3 days

## Job To Be Done

As admin, I want to credit a parent's wallet against a bank transfer they've made.

## Acceptance Criteria

- AC1: `bank_transfer_pending` table captures pending notifications (manual entry by admin or future bank API).
- AC2: Admin matches a transfer to a parent → confirms → `wallet.post(topup)` with `source='bank:manual'`.
- AC3: Parent SMS-stub notified.

## Technical Notes

No automated bank reconciliation in P1; manual entry only.

## Dependencies
