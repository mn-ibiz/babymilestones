# P1-E02-S05 — Data export for a parent's record

**Status:** Ready for development
**Phase:** P1 — Foundation + Wallet + Parent Account
**Epic:** P1-E02 — Parent & Child Registry
**Estimate:** 1–3 days

## Job To Be Done

As a parent, I want to download everything you have on me and my children, in line with Kenya's Data Protection Act.

## Acceptance Criteria

- AC1: "Export my data" button on parent profile → ZIP with JSON for parent, children, bookings, wallet ledger, receipts.
- AC2: Generation is async (>5s); SMS-stub sends a download link, valid 7 days, single-use.
- AC3: Audit logged.

## Technical Notes

Job runs in `apps/jobs`. ZIP stored at signed-URL S3-equivalent.

## Dependencies

- P1-E03 (ledger)
- P1-E09 (SMS stub).
