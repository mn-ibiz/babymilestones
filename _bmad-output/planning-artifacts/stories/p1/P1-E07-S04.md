# P1-E07-S04 — VAT / tax flag per service

**Status:** Ready for development
**Phase:** P1 — Foundation + Wallet + Parent Account
**Epic:** P1-E07 — Service Catalogue & Pricing
**Estimate:** 1–3 days

## Job To Be Done

As accountant, I want each service to declare its tax treatment so receipts and reports show VAT correctly.

## Acceptance Criteria

- AC1: `services.tax_treatment` ENUM (`vat_inclusive`, `vat_exclusive`, `vat_exempt`, `zero_rated`).
- AC2: Receipt engine (P1-E08) shows line-tax accordingly.
- AC3: Default `vat_exempt` (KRA registration deferred).
