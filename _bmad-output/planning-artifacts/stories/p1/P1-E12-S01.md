# P1-E12-S01 — Home page

**Status:** Ready for development
**Phase:** P1 — Foundation + Wallet + Parent Account
**Epic:** P1-E12 — Marketing & Landing (public route group)
**Estimate:** 1–3 days

## Job To Be Done

As a first-time visitor, I want to understand Baby Milestones in 8 seconds and tap to sign up.

## Acceptance Criteria

- AC1: Hero: real photo of a real child + headline + visible CTA ("Top up & book").
- AC2: 4-icon unit strip below hero (Play / Talent / Salon / Toy Shop — Toy Shop links out to the standalone WooCommerce site).
- AC3: No carousel.
- AC4: SSR for SEO; LCP < 2s on 3G.
