# P1-E12-S02 — Per-unit pages

**Status:** Ready for development
**Phase:** P1 — Foundation + Wallet + Parent Account
**Epic:** P1-E12 — Marketing & Landing (public route group)
**Estimate:** 1–3 days

## Job To Be Done

As a visitor, I want a page per unit with what it offers and a "Book now" CTA.

## Acceptance Criteria

- AC1: Pages: `/play`, `/talent`, `/salon`, `/events`, `/coaching`. The Toy Shop is an **external link** to the standalone WooCommerce site (no `/shop` route in this app).
- AC2: Each: photo, short copy, examples, "Book now" CTA → signup if not logged in.
- AC3: Content sourced from MDX or DB (admin-editable in P5 polish).

## Definition of Done

Standard DoD applies. A story is **Done** when:
1. Code reviewed by another engineer.

6. PM + designer walked through the staging build for the affected surface.
7. No regression in `e2e/` suite.
