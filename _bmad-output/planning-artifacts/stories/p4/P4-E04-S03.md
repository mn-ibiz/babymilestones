# P4-E04-S03 — Print packing slip

**Status:** Ready for development
**Phase:** P4 — WooCommerce Sync + Events Ticketing
**Epic:** P4-E04 — POS ↔ WooCommerce Sync
**Estimate:** 1–3 days

## Job To Be Done

As a packer, I want to print a packing slip per WooCommerce order so I can pack and dispatch it.

## Acceptance Criteria

- AC1: "Print packing slip" button on order card.
- AC2: Slip lists: Woo order number, customer name + phone, shipping address (from Woo), delivery method, line items + qty, customer note / special instructions.
- AC3: Uses system default printer (Decision 13).
- AC4: Slip is rendered from the local `wc_orders` mirror — no live Woo call required at print time.

## Technical Notes

- Reuses the receipt rendering pipeline from P1-E08 where possible; packing slip is a distinct template (no price totals required, but qty is mandatory).

## Tests

- Unit: template renders with required fields; missing address falls back to "Pickup in store" note.

## Dependencies

- S01
- P1-E08 (receipt engine / PDF render)
