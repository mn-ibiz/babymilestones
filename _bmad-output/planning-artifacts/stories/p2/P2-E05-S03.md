# P2-E05-S03 — Redemption at parent checkout

**Status:** Ready for development
**Phase:** P2 — Bookings, Subscriptions, POS, Loyalty Redemption
**Epic:** P2-E05 — Loyalty Redemption UI + Engine
**Estimate:** 1–3 days

## Job To Be Done

As a parent, I want to use my points to reduce my booking or shop bill.

## Acceptance Criteria

- AC1: At booking confirmation (in the custom platform), a toggle: "Use X points (save KES Y)". (WooCommerce online purchases are out of scope for loyalty — Decision 37.)
- AC2: Toggle on → applies points as a wallet credit equal to `points × redeem_rate`, deducts from the bill.
- AC3: Cannot redeem more points than current balance; cannot redeem points already on a pending settlement.
- AC4: Redemption writes a `loyalty_ledger` debit + a `wallet_ledger` credit + the booking debit applies normally.
