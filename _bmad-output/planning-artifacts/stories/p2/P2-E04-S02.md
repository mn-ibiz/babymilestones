# P2-E04-S02 — Product catalogue read for POS

**Status:** Ready for development
**Phase:** P2 — Bookings, Subscriptions, POS, Loyalty Redemption
**Epic:** P2-E04 — POS App (in-store mode)
**Estimate:** 1–3 days

## Job To Be Done

As a cashier, I want to search or scan a product and add it to a sale.

## Acceptance Criteria

- AC1: Barcode scanner input auto-focused; on enter → matches `products.sku` or `products.barcode`.
- AC2: Search by name with debounce; results show price, stock.
- AC3: Out-of-stock products greyed out (sale blocked at checkout).

## Technical Notes

Uses `packages/catalog`. Catalogue itself created in P4-E01 — for P2 ship a minimal stub seed product set.
