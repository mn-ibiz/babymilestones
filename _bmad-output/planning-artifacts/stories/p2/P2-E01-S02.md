# P2-E01-S02 — Parent browses available slots for a service

**Status:** Ready for development
**Phase:** P2 — Bookings, Subscriptions, POS, Loyalty Redemption
**Epic:** P2-E01 — Booking Engine
**Estimate:** 1–3 days

## Job To Be Done

As a parent, I want to see this week's available Play / Talent slots so I can book what fits.

## Acceptance Criteria

- AC1: Service detail page shows a 7-day grid with available slots + remaining capacity.
- AC2: Slots filtered to those the child's age fits (uses `services.age_min` / `age_max`).
- AC3: Past slots greyed out; today's earlier slots disabled.
- AC4: Loads ≤500ms p95.

## Technical Notes

Indexed query on `session_slots`. `apps/platform/app/(app)/book/[service]/page.tsx`.
